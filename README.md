# TrabaHome Platform

## Project Description
TrabaHome is a web-based service platform that connects homeowners with verified skilled workers.

The system uses AI-assisted identity verification and blockchain-backed audit logging to improve trust and security.

## System Architecture
React Web App → Node.js Backend → MongoDB Database

External services:
- AI verification
- Blockchain audit logging
- Google Maps API
- SMS notifications

## Setup Instructions
1. Install Node.js
2. Clone repository
3. Run npm install
4. Start backend server
5. Start frontend app
